# Simple Addition Calculator

A simple web-based calculator built with HTML, CSS, and JavaScript.

This project allows users to enter two numbers (Operand 1 and Operand 2) and calculate the result dynamically using JavaScript.

---

## Features

- Two numeric input fields
- Calculate button
- Dynamic result display
- Basic input validation
- Clean and beginner-friendly UI

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

---

## How It Works

1. Enter a number into the first input field
2. Enter another number into the second input field
3. Click the **Calculate** button
4. The result will be displayed instantly on the screen

---

## Example

```txt
Input A: 5
Input B: 10

Result: 15
```
